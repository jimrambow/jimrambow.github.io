GitHub

This program incorporates join tables and a work-flow.

Online Menu Wizard

GitHub

This was my final project. I wanted to make an application which made it easy for large organizations that have many restuarants. It ended up as a wizard to create your own organization and add restuarants and menus to the restuarants. As you make the new menu, you can add items to the menu as you work. After you have completed the menu, it may be printed in a simple click of a button. Has a choice of different types of payment plans using Stripe.

Primary Technologies

  • Ruby on Rails
  • Javascript
  • jQuery
  • JSON
  • MySQL
  • Stripe

Other Tools

  • GitHub
  • Twitter Bootstrap
  • WickedWizard (Ruby Gem)
